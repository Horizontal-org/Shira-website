# Добро пожаловать в справочный центр Shira!

Shira — это инструмент для обучения противодействию фишингу. С помощью Shira
можно научить любого человека распознавать фишинговые атаки, осуществляемые
через электронную почту, СМС и мессенджеры.

Shira предлагает симуляции фишинговых атак, в контролируемой учебной среде ,
максимально приближенные к тем, с которыми ваша команда столкнется в реальной
жизни. Система Shira основана на [исследованиях, которые
показали](/phishing-quizzes), что в плане развития навыков защиты от фишинга
обучение оказывается гораздо более эффективным, чем отправка симулированных
фишинговых писем на почтовые ящики учащихся.

В этом справочном центре вы найдете рекомендации по использованию Shira для
обучения вашей команды или сообщества. Если вы не нашли здесь ответы на
некоторые из своих вопросов, [свяжитесь с нами!](/contact)

### Пространство Shira {#shira-space}
[Shira Space](/spaces) предназначен для команд и организаций и предлагает такие
расширенные функции, как:
- создание собственных вопросов для викторины с содержанием, специально
  подобранным для вашей команды.
- Вы можете установить для теста настройки конфиденциальности: «Общедоступный»
  (доступен любому пользователю по URL-адресу теста) или «Частный» (доступен
  только учащимся, зарегистрированным в вашей организации). Частные тесты будут
  доступны в ближайшее время!
- отслеживание результатов пользователей, чтобы понять, в чем они могут улучшить
  свои навыки.
