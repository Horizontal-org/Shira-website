# Добро пожаловать в справочный центр Shira!

Shira — инструмент для обучения противодействию фишингу. С помощью Shira можно
научить любого человека распознавать фишинговые атаки, осуществляемые через
email, СМС и мессенджеры.

Shira предлагает симуляции фишинговых атак в контролируемой учебной среде ,
максимально приближённые к тем, с которыми ваша команда столкнётся в реальной
жизни. Shira основана на [исследованиях, которые показали](/phishing-quizzes),
что для развития навыков защиты от фишинга обучение гораздо эффективнее рассылки
симулированных фишинговых писем на почтовые ящики учащихся.

Наш Справочный центр предлагает рекомендации по использованию Shira для обучения
вашей команды или сообщества. Не нашли ответ на свой вопрос? [Свяжитесь с
нами!](/contact)

### Пространство Shira {#shira-space}
[Пространство Shira](/spaces) предназначено для команд и организаций и
предлагает расширенные функции, например:
- создание собственных вопросов теста, адаптированных для вашей команды;
- выбор между открытым (доступен любому пользователю по ссылке) или закрытым
  (только для вашей команды) режимами (закрытые тесты станут доступны в
  ближайшее время);
- отслеживание результатов пользователей, чтобы понять, как они могут улучшить
  свои навыки.
