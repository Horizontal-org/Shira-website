# Собственный хостинг

Shira — [продукт с открытым исходным
кодом](https://github.com/Horizontal-org/shira), распространяется по лицензии
MIT 2.0. Мы поддерживаем облачную версию Shira для наших партнеров, но любой
желающий может установить Shira на свой сервер.

Если вы хотите разместить Shira на своём сервере, пожалуйста, [свяжитесь с
нами](/contact). Впрочем, пользователи, у которых есть опыт самостоятельного
хостинга, могут справиться и сами:
[https://github.com/Horizontal-org/shira](https://github.com/Horizontal-org/shira).
В ближайшее время мы опубликуем руководство по настройке Shira на своём
хостинге.
