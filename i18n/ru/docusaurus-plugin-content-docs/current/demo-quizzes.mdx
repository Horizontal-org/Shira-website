# Готовые викторины

Любой желающий может бесплатно воспользоваться готовыми тестами Shira, чтобы
развить свои навыки по борьбе с фишингом:

<Link 
  className={classNames(global.buttonDark)} 
  to="https://quiz.shira.app/"
>Try a ready-made quiz now</Link>

Готовые викторины позволяют в определенной степени настраивать их, в том числе
давая пользователям возможность:
* Выберите приложения, которыми они пользуются каждый день.
* Выберите «сферу деятельности», чтобы содержание теста по фишингу было для них
  более актуальным.
* Выберите нужный язык из списка поддерживаемых языков
* Отобразите их имя и адрес электронной почты в викторине, чтобы каждый адрес и
  сообщение выглядели максимально похожими на то, с чем они столкнутся в
  реальной жизни.

Готовые тесты — отличный способ начать развивать навыки защиты от фишинга.
Однако обучение по противодействию фишингу наиболее эффективно, когда тесты
адаптированы к конкретной учебной цели и включают реальные сценарии, актуальный
контекст, а также уникальные особенности организации, стиль изложения и уровень
сложности. Чтобы добиться такой адаптации, вам может понадобиться создать [Shira
Space](/spaces).

Если вам нужна помощь в том, чтобы определить, подходит ли создание такого
пространства для вашей организации, [свяжитесь с нами](/contact)!
