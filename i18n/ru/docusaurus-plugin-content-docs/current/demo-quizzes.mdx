# Готовые тесты

Любой желающий может бесплатно попробовать готовые тесты Shira, чтобы развить
свои навыки по борьбе с фишингом.

<QuizLink className={classNames(global.buttonDark)}>Пройти готовый
тест</QuizLink>

Готовые тесты можно немного настраивать, в частности:
* выбрать одно из привычных приложений;
* выбрать «сферу деятельности», чтобы содержание теста было релевантным;
* выбрать нужный язык из списка поддерживаемых языков;
* показывать имя и email в тесте, чтобы сгенерированные сообщения были
  максимально похожи на те, которые мы получаем в реальной жизни.

Готовые тесты — отличный способ начать развивать навыки защиты от фишинга.
Однако обучение по противодействию фишингу наиболее эффективно, когда тесты
адаптированы к конкретной учебной цели и включают реальные сценарии, актуальный
контекст, а также уникальные особенности организации, стиль изложения и уровень
сложности. Чтобы добиться такой адаптации, вам может понадобиться создать
[пространство Shira](/spaces).
