# Результаты

В разделе «Результаты» можно узнать, прошли ли пользователи ваш тест и насколько
хорошо они с ним справились.

В разделе «Результаты» вы можете увидеть следующую информацию: **Число
завершенных тестов** (сколько раз учащиеся прошли тест).
* Число завершенных открытых тестов не всегда равно числу учащихся, прошедших
  тест: учащиеся анонимны, значит, один человек может пройти тест несколько раз.
* Закрытый тест зарегистрированный учащийся может пройти только один раз.

**Средний балл**\
Средний балл всех учащихся, которые прошли тест.

**Коэффициент завершения** [скоро!]\
Процент учащихся, которые прошли тест, от общего числа учащихся, которым был
предложен этот тест. Коэффициент завершения доступен только для закрытых тестов.

**Результаты по вопросам** [скоро!]\
Средний балл, полученный всеми учащимися по каждому конкретному вопросу в тесте.

**Результаты по учащимся** [скоро!]\
Список зарегистрированных учащихся, которые прошли тест, и их результаты.
Результаты по учащимся доступны только для закрытых тестов.

:::note
Если вы хотели бы видеть в результатах что-то ещё, [свяжитесь с
нами](/contact)!
:::
