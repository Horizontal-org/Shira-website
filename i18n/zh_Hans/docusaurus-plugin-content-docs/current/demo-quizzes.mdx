# 现成测验

任何人都可以免费参加 Shira 现成测验，以提升防钓鱼技能：

<QuizLink className={classNames(global.buttonDark)}>Try a ready-made quiz
now</QuizLink>

现成测验提供基础级别的自定义选项，包括：
* 选择日常使用的应用。
* 选择"工作领域"，使钓鱼测验内容更贴合用户的实际情况。
* 从支持的语言中选择偏好的语言
* 在测验中显示用户姓名和电子邮箱，以便每则邮件和消息尽可能接近用户面临的实际情景。

现成测验是一种很好的防钓鱼培训方式。最有效的防钓鱼培训需要：根据具体学习目标量身定制测验，融入真实场景和相关背景信息，同时包含组织特有细节、风格和复杂性。为了实现这种定制，您需要创建
[Shira 空间](/spaces)。

如果需要帮助，评估空间是否适合您的组织，请[联系我们](/contact)！
