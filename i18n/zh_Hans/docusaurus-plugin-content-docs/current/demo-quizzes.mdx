# 现成测验

任何人都可以免费参加 Shira 现成测验，来培养反钓鱼技能：

<Link 
  className={classNames(global.buttonDark)} 
  to="https://quiz.shira.app/"
>Try a ready-made quiz now</Link>

现成测验提供基础级别的自定义选项，包括：
* 选择用户日常使用的应用。
* 选择"工作领域"，使钓鱼测验内容更贴合用户的实际情况。
* 从支持的语言中选择偏好的语言。
* 在测验中显示用户姓名和电子邮箱，使每则邮件和消息尽可能接近用户面临的实际情景。

现成测验是一种很好的反钓鱼培训方式。不过，最有效的反钓鱼培训则需要：根据具体学习目标量身定制测验，融入真实场景和相关背景信息，同时包含组织特有细节、风格和复杂性。为了实现这种定制，您需要创建
[Shira 空间](/spaces)。

如果需要帮助，评估空间是否适合您的组织，请[联系我们](/contact)！
