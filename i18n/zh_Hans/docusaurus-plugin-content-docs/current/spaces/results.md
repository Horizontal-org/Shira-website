# 结果

“结果”部分可查看用户是否完成测验以及测验表现情况。

结果包含以下信息：**已完成测验的数量** 这是学员参加测验的次数。
* 对于公开测验，匿名成员可多次参加测验，因此已完成测验的数量未必等于参加测验的学员人数。
* 对于私有测验，注册学员每个测试只可参加一次。

**平均分数**\
表示所有参加测验的学员获得的平均分数。

**完成率** [即将推出]\
在所有被分配该测验的学员中，提交测验的学员所占的百分比。完成率仅适用于私有测验。

**题目结果** [即将推出]\
对于测验中每道具体题目，所有学员获得的平均分数。

**学员结果** [即将推出]\
完成测验的注册学员列表及其测验分数。学员结果仅适用于私有测验。
