# 自托管

Shira 开源[open-source](https://github.com/Horizontal-org/shira) 并基于 MIT 2.0
许可协议分发。我们为合作伙伴维护 Shira 云端实例，不过任何人都可以自托管自己的实例。

如果想要自托管
Shira，请[联系我们](/contact)。经验丰富的自托管用户可直接操作：[https://github.com/Horizontal-org/shira](https://github.com/Horizontal-org/shira)。自托管指南将很快发布。
