# Quiz prêts à l'emploi

N'importe qui peut utiliser gratuitement les quiz prêts à l'emploi de Shira pour
renforcer ses compétences contre le phishing :

<QuizLink className={classNames(global.buttonDark)}>Try a ready-made quiz
now</QuizLink>

Les quiz prêts à l'emploi offrent un niveau de personnalisation de base,
permettant notamment aux utilisateurs de :
* Sélectionnez les applications qu'ils utilisent quotidiennement.
* Choisissez un “domaine d'activité” pour rendre le contenu du quiz de phishing
  plus pertinent pour eux.
* Sélectionnez leur langue préférée parmi les langues prises en charge
* Affichez leur nom et leur email à l'intérieur du quiz, afin que chaque e-mail
  et message ressemble le plus possible à ce qu'ils rencontreront dans la vie
  réelle.

Les quiz prêts à l'emploi sont un excellent moyen de commencer à acquérir des
compétences contre le phishing. Cependant, la formation contre le phishing est
plus efficace lorsque les quiz sont adaptés à un objectif d'apprentissage
spécifique et intègrent des scénarios réels, un contexte pertinent ainsi que les
détails, le ton et le niveau de complexité propres à une organisation. Pour
parvenir à ce type de personnalisation, vous devrez peut-être créer un [Espace
Shira](/spaces).

Si vous avez besoin d'aide pour déterminer si la création d'un espace est la
bonne solution pour votre organisation, [contactez-nous](/contact) !
