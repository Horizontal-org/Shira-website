# Cuestionarios predefinidos

Cualquiera puede usar los cuestionarios predefinidos de Shira de forma gratuita
para desarrollar sus habilidades anti-phishing:

<Link 
  className={classNames(global.buttonDark)} 
  to="https://quiz.shira.app/"
>Try a ready-made quiz now</Link>

Los cuestionarios predeterminados proporcionan un nivel básico de
personalización, incluyendo al permitirle a sus personas usuarias:
* Selecciona las aplicaciones que utilizan cotidianamente.
* Elige un "campo de trabajo" para hacerles el contenido de phishing más
  relevante.
* Selecciona su idioma preferido entre los idiomas admitidos
* Muestra su nombre y correo electrónico dentro del cuestionario, para hacer que
  cada correo y mensaje se parezca lo mejor posible a lo que enfrentarán en la
  vida real.

Los cuestionarios listos para usar son una excelente manera de empezar a
desarrollar habilidades anti-phishing. Sin embargo, la capacitación
anti-phishing es más efectiva cuando los cuestionarios son diseñados con una
meta de aprendizaje especifica e incluyen escenarios de la vida real, contexto
relevante y detalles, tono y nivel de complejidad únicos de la organización.
Para alcanzar este nivel de personalización, puedes necesitar crear un [Espacio
de Shira](/spaces).

Si necesitas ayuda para evaluar si crear un espacio es el camino adecuado para
tu organización, ¡[contáctanos](/contact)!
