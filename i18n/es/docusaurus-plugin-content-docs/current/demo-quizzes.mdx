# Ready-made quizzes

Cualquiera puede usar los cuestionarios predefinidos de Shira de forma gratuita
para desarrollar sus habilidades anti-phishing:

<Link 
  className={classNames(global.buttonDark)} 
  to="https://quiz.shira.app/"
>Try a ready-made quiz now</Link>

Los cuestionarios predeterminados proporcionan un nivel básico de
personalización, como:
* Selecciona las aplicaciones que utilizan cotidianamente.
* Elige un "campo de trabajo" para hacerles el contenido de phishing más
  relevante.
* Selecciona su idioma preferido entre los idiomas admitidos.
* Display their name and email inside the quiz, to make each email and message
  look as similar as possible to what they will face in real life.

Ready-made quizzes are a great way to start building anti-phishing skills.
However, anti-phishing training is most effective when quizzes are tailored to a
specific learning goal and incorporates real-life scenarios, relevant context,
and an organization's unique details, tone, and complexity level. To achieve
this type of customization, you may need to create a [Shira Space](/spaces).

If you need help assessing whether creating a space is the right path for your
organization, [contact us](/contact)!
