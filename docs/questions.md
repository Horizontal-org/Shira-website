Quiz questions are where the learning happens. This is where you show your learners an email or message, and ask them: does this look like phishing? 

In each quiz, you can include as many questions as wish.

### Question preferences {#preferences}
When you create a question, you must: 
- Name the question: this name, like "Invitation to conference", is only visible by space admins and meant to help you recognize the questions you create. Learners never see a question's name. 
- Tell Shira whether the email or message you're creating is *phishing* or a *legitimate* question. This will allow Shira to determine whether a user's response is correct.
